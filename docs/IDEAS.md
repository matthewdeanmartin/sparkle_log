# Sparkline Ideas

## 1. Braille
Uses Unicode Braille patterns for a tactile, accessible look.
```
symbols = ["⠁", "⠂", "⠃", "⠄", "⠅", "⠆", "⠇", "⠈", "⠉", "⠊"]
example: ⠁⠃⠅⠆⠇⠈⠊⠙
```
